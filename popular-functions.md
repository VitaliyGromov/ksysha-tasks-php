Ниже представлена подборка самых востребованных функций PHP для работы с основными типами данных: строками, массивами, числами и булевыми значениями. Эти функции встречаются в большинстве проектов и реально экономят время.

## 🔤 Строки (strings)

| Функция | Описание | Я часто использую |
|---------|----------| ------------------|
| `strlen()` | Возвращает длину строки. | ✅ |
| `trim()` | Удаляет пробелы (и другие символы) из начала и конца строки. | ✅ |
| `substr()` | Возвращает подстроку. | Редко капец |
| `strpos()` | Ищет позицию первого вхождения подстроки (чувствительна к регистру). | Редко капец | 
| `str_replace()` | Заменяет все вхождения строки поиска на строку замены. | ✅ |
| `explode()` | Разбивает строку на массив по разделителю. | ✅ |
| `implode()` / `join()` | Объединяет элементы массива в строку. | ✅ Использую implode() |
| `strtolower()` / `strtoupper()` | Приводит строку к нижнему/верхнему регистру. | ✅ |
| `ucfirst()` | Преобразует первый символ строки в верхний регистр. | ✅ |
| `htmlspecialchars()` | Преобразует специальные символы в HTML-сущности (защита от XSS). | Ни разу не применял, в Laravel не пригодится особо | 
| `sprintf()` | Возвращает отформатированную строку. | Ни разу не использовал | 
| `mb_strlen()`, `mb_substr()` и др. | Многобайтовые аналоги (для UTF-8 и других кодировок). | ✅ mb_strlen() использовал |

## 📦 Массивы (arrays)

| Функция | Описание |
|---------|----------|
| `count()` | Подсчитывает количество элементов массива. |
| `array_push()` / `array_pop()` | Добавляет элемент в конец / извлекает элемент из конца массива. |
| `array_shift()` / `array_unshift()` | Извлекает элемент из начала / добавляет элемент в начало массива. |
| `array_merge()` | Сливает один или несколько массивов. |
| `array_diff()` | Вычисляет расхождение массивов. |
| `array_intersect()` | Вычисляет пересечение массивов. |
| `array_keys()` | Возвращает все или некоторое подмножество ключей массива. |
| `array_values()` | Возвращает все значения массива (переиндексирует). |
| `in_array()` | Проверяет, существует ли значение в массиве. |
| `array_search()` | Ищет значение в массиве и возвращает первый соответствующий ключ. |
| `sort()`, `rsort()` | Сортирует массив по значениям (по возрастанию/убыванию), сбрасывая ключи. |
| `asort()`, `arsort()` | Сортирует массив с сохранением связи ключ–значение. |
| `ksort()`, `krsort()` | Сортирует массив по ключам. |
| `array_map()` | Применяет callback-функцию ко всем элементам массива. |
| `array_filter()` | Фильтрует элементы массива с помощью callback-функции. |
| `array_reduce()` | Итеративно сворачивает массив к единственному значению. |
| `array_slice()` | Извлекает срез массива. |
| `array_splice()` | Удаляет часть массива и заменяет её чем-либо. |
| `array_reverse()` | Возвращает массив с элементами в обратном порядке. |
| `array_column()` | Возвращает значения одного столбца многомерного массива. |
| `list()` (или `[]`) | Присваивает переменным значения, как если бы массив был набором значений. |

## 🔢 Числа (int / float)

| Функция | Описание |
|---------|----------|
| `intval()` | Возвращает целочисленное значение переменной. |
| `floatval()` / `doubleval()` | Возвращает значение переменной как число с плавающей точкой. |
| `round()` | Округляет число с плавающей точкой. |
| `ceil()` | Округляет дробь вверх (до ближайшего большего целого). |
| `floor()` | Округляет дробь вниз (до ближайшего меньшего целого). |
| `abs()` | Возвращает абсолютное значение числа. |
| `pow()` | Возводит в степень. |
| `sqrt()` | Извлекает квадратный корень. |
| `rand()` / `mt_rand()` | Генерирует случайное целое число. |
| `number_format()` | Форматирует число с разделением групп тысяч и десятичных знаков. |
| `is_int()` / `is_float()` | Проверяет, является ли переменная целым числом / float. |
| `intdiv()` | Целочисленное деление (возвращает целую часть). |
| `fmod()` | Возвращает остаток от деления чисел с плавающей точкой. |

## ✅ Булевы типы (boolean)

| Функция / Конструкция | Описание |
|----------------------|----------|
| `boolval()` | Возвращает булево значение переменной. |
| `is_bool()` | Проверяет, является ли переменная булевой. |

## Json
Вот это реально нужно использовать и знать. 

### json_encode()
Функция для преобразования значения в json строку. Например, если передать число, то на выходе будет строка
```
var_dump(json_encode(23));
```
```
string(2) "23"
```

Ну и вот так можно запилить массив в json (Это самое частое, для чего она будет использоваться)
```
var_dump(json_encode(['a' => 'b']));
```
```
string(9) "{"a":"b"}"
```

### json_decode()
Функция для декодирования json. При разработке часто будет использоваться для преобразования json строки в ассоциативный массив
```
var_dump(json_decode('{"a":"b"}', true)); 
```
Выведет
```
['a' => 'b']
```
> 💡 **Совет:** для строк в реальных проектах, где используется UTF-8, предпочитайте `mb_`-функции, чтобы избежать проблем с кодировкой. Для массивов часто бывает удобен `array_column()` при работе с результатами SQL-запросов. Для чисел не забывайте про `number_format()` для красивого вывода.
